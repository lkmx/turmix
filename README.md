# Turmix

Abstraction layer for dealing with data powered by Spark.

## Set up

1. Clone the repo

```sh
git clone https://github.com/lkmx/turmix.git
cd turmix
```

2. Install turmix in the local maven repo

```sh
mvn install
```

3. Add it to your project

```xml
<dependency>
    <groupId>io.lkmx</groupId>
    <artifactId>turmix</artifactId>
    <version>0.0.1</version>
</dependency>
```
